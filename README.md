# Vovinam Võ 2 - Luyện Vấn Đáp

Ứng dụng web hỗ trợ tự luyện vấn đáp Vovinam Việt Võ Đạo cấp Võ 2.

Người học có thể bốc thăm câu hỏi ngẫu nhiên theo hình thức thi thực tế, nghe câu hỏi bằng giọng nói, trả lời bằng micro và tự đối chiếu với đáp án mẫu.

## Tính năng

* Bốc thăm ngẫu nhiên bộ câu hỏi theo quy định thi vấn đáp.
* Hỗ trợ câu hỏi đáp dài và đáp ngắn.
* Đọc câu hỏi bằng giọng nói (Text-to-Speech).
* Ghi nhận câu trả lời bằng giọng nói (Speech Recognition, tùy trình duyệt hỗ trợ).
* Tự đánh giá mức độ đầy đủ của câu trả lời dựa trên các ý chính.
* Hiển thị đáp án mẫu để đối chiếu.
* Thống kê kết quả sau khi hoàn thành bài luyện tập.
* Giao diện tối ưu cho điện thoại và máy tính.

## Công nghệ sử dụng

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Web Speech API

## Chạy trên máy tính

Tải project về và mở file:

```text
index.html
```

bằng trình duyệt Chrome, Edge hoặc Safari.

## Triển khai bằng GitHub Pages

1. Tạo repository mới trên GitHub.
2. Upload toàn bộ source code.
3. Vào:

```text
Settings
→ Pages
```

4. Tại mục Build and deployment:

```text
Source: Deploy from a branch
Branch: main
Folder: /(root)
```

5. Nhấn Save.

Sau khi triển khai thành công, website sẽ có địa chỉ:

```text
https://<github-username>.github.io/vovinam-vo2-van-dap/
```

## Lưu ý

* Tính năng nhận diện giọng nói phụ thuộc vào trình duyệt.
* Chrome và Edge trên máy tính cho trải nghiệm tốt nhất.
* Một số phiên bản Safari trên iPhone có thể không hỗ trợ đầy đủ Speech Recognition.

## Nguồn dữ liệu

Nội dung câu hỏi và đáp án được biên soạn dựa trên tài liệu lý thuyết Vovinam Việt Võ Đạo cấp Võ 2 phục vụ mục đích học tập và ôn luyện.

## Giấy phép

Dự án được chia sẻ cho mục đích học tập, nghiên cứu và tự rèn luyện cá nhân.
